# Wireshark DNS Traffic Analysis Project

## Objective
Capture and analyze DNS traffic using Wireshark.

## Tools Used
- Wireshark
- macOS

## Steps Performed
- Captured live network traffic
- Applied DNS filter (dns)
- Analyzed DNS queries and responses

## Key Findings
- Observed DNS requests for domains like google.com
- Identified A (IPv4) and AAAA (IPv6) records
- Saw communication between local device and DNS servers
- Understood how DNS resolves domain names into IP addresses
